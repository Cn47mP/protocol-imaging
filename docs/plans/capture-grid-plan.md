# 采集网格规划草本 v0.2

> 更新：2026-05-29

## 前置校准
1. 进入 AIC 模式，相机推到最高，转至垂直俯视
2. 截图 → 数可见格子数（水平×垂直）
3. 填入 docs/capture-plan.md 的估算值

## 预期参数（校准前暂估）
- 单帧 1920×1080 覆盖：~8×5 格
- 满扩建四号谷地核心区：~36×40 格
- 网格：6 列 × 10 行 = 60 帧
- 重叠率：20%
- 移动方式：Z 字逐行扫描

## 当前实现
- `AutoCapturer` 已支持 Z 字蛇形扫描
- 预设网格：small 2×2、medium 3×3、large 4×4、xlarge 5×5
- 每帧采集后执行模糊检测，低质量帧会跳过
- GUI 已提供自动检测窗口、预设选择、进度显示、取消采集
- 当前预设仍偏演示/原型，未按真实满扩建 AIC 面积校准

## 下一步
1. 用真实游戏截图校准单帧可见格数
2. 将估算的 6×10 / 20% 重叠转成可配置采集方案
3. 在 GUI 中增加自定义行列数、移动距离、重叠率
4. 保存校准配置，避免每次重新调参
